## 工具简介

本工具是一个**无需安装、无需服务器**的单 HTML 文件工具。在浏览器中直接打开即可使用，通过导入本地 `.xlsx` / `.xls` / `.xlsm` 格式的 AI 评测结果表格，实现：

- 可视化展示每条数据在业务流各节点（知识快系统 → 慢系统 → 结果）中的链路状态
- 快速定位哪些节点发生了错误（`is_wrong = true`）
- 统计各节点错误数量占比，辅助定位问题根因
- 支持多维度字段筛选和列管理

---

## 快速开始

1. 用浏览器（Chrome / Edge 推荐）打开 `result_parser_topology_v0_10_regenerated.html`
2. 点击左上角 **「打开本地 Excel」** 按钮，选择评测结果表格（如 `20260707.xlsx`）
3. 导入后页面自动展示：
   - 顶部：**错误根因分布统计柱状图**
   - 下方：**可筛选数据表格**
4. 点击任意数据行，弹出**单行详情面板**，可查看业务流拓扑和各节点内容

## 支持的表格字段

工具默认优先识别以下字段（列名需与表头完全一致）：

| 字段名 | 类型 | 用途说明 |
|--------|------|----------|
| `query` | 文本 | 用户输入问题 |
| `sessionId` | 文本 | 会话 ID，在表格中展示 |
| `出错原文` | 文本 | 模型最终输出内容，驱动「结果」节点状态 |
| `intent` | JSON | 意图识别结果，驱动「任务快系统」节点 |
| `webSearch` | JSON | 检索路径结果，驱动「websearch」节点 |
| `queryUnstandingRouterType` | JSON | 意图分流结果，驱动「意图分流」节点 |
| `知识库` | JSON | KAG 知识检索结果，驱动「KAG知识」节点 |
| `运营话术` | JSON | KAG 直出内容，驱动「KAG直出」节点 |
| `网页知识` | JSON | 网页知识检索结果，驱动「网页知识」节点 |
| `垂域.md` | JSON | 垂域上下文内容，驱动「垂域.md」节点 |
| `模型问题` | JSON | UniLLM 推理结果，驱动「UniLLM」节点，支持 is_wrong |
| `评测结果` | 标签 | 人工或自动评测结论，表格中以彩色标签展示 |
| `相关性` | 标签 | 评测维度：相关性 |
| `正确性` | 标签 | 评测维度：正确性 |
| `信息量` | 标签 | 评测维度：信息量 |
| `备注` | 文本 | 备注信息 |

> **注意**：表格中不存在的字段会被自动跳过，不影响其他字段的展示。`评测结果` 等字段暂时缺失时，工具逻辑保持不变，待数据补充后自动生效。

## 业务流拓扑图

点击数据行后，详情弹窗左侧展示业务流拓扑图，将一条数据在系统各节点的命中情况可视化：

```
用户 query
   ├──→ 【意图分流】(queryUnstandingRouterType)
   │         ├──→ 【KAG直出】(运营话术)
   │         └──→ 【websearch】
   │                   ├──→ 【KAG知识】(知识库)
   │                   └──→ 【网页知识】(网页知识)
   │                              └──→ 【UniLLM】(模型问题) ──→ 【结果】(出错原文)
   └──→ 【任务快系统】(intent) ──────────────↗
              【垂域.md】(垂域.md) ──────────↗
```

### 节点着色规则

| 颜色 / 标记 | 含义 |
|-------------|------|
| 🟢 绿色 | 字段有值，节点正常命中 |
| ⚫ 灰色 | 字段为空，节点未命中 |
| 🔵 蓝色边框 | 当前选中节点 |
| ❌ 红叉（`X`） | 字段 JSON 中 `is_wrong = true` |
| `评测模型未评测` | `is_time_out = true` |
| `empty` | `is_empty = true` |
| 中性色（无标绿） | 任务快系统的 intent 属于 QA / KAG / CHAT / DISPLAY_NLU_THINK#CHAT |
| `字段不存在` | 当前表头中不包含该字段 |

点击拓扑图中的任意节点，右侧详情区自动切换展示该节点的**问题分析（reason）** 和**内容（content）**。

## 单行详情面板

点击数据表格中任意一行，弹出详情面板，包含两个区域：

**① 业务流拓扑图**（左/上方）
- 可视化当前行数据的全链路节点状态
- 点击节点切换右侧内容显示

**② 节点内容区**（右/下方）

| 区块 | 说明 |
|------|------|
| 问题分析 | 对应字段 JSON 中的 `reason` 子字段 |
| 内容 | 对应字段 JSON 中的 `content` 子字段，或字段原始值 |
| 一键复制 | 复制当前节点「内容」区的完整文本 |

## 字段筛选与列管理

### 字段级筛选
每列表头下方有独立筛选输入框，输入关键词后实时过滤当前列，支持同时对多列筛选。支持中文输入法（使用 compositionend 事件，不会在拼音中途触发）。

### 全局搜索
工具栏「搜索当前表格内容」输入框，跨所有可见列进行模糊匹配。

### 列管理
点击「选择列」按钮，勾选或取消需要展示的字段。默认展示 DEFAULT_COLUMNS 中已存在于表头的字段。

### 清空筛选
点击「清空筛选」按钮，一键重置所有字段筛选条件。

## JSON 字段解析说明

`intent`、`webSearch`、`queryUnstandingRouterType`、`知识库`、`运营话术`、`网页知识`、`垂域.md`、`模型问题` 这 8 个字段的值通常是 JSON 对象或 Python dict 格式字符串，工具会自动尝试解析并提取以下子字段：

| 子字段 | 说明 |
|--------|------|
| `content` | 节点实际输出内容，在详情面板「内容」区展示 |
| `reason` | 问题分析描述，在详情面板「问题分析」区展示 |
| `is_wrong` | 布尔值，`true` 时节点显示红叉 ❌ |
| `is_time_out` | 布尔值，`true` 时节点显示「评测模型未评测」标签 |
| `is_empty` | 布尔值，`true` 时节点显示「empty」标签 |

### 兼容的格式

工具依次尝试以下解析策略，直到成功为止：

1. 标准 JSON（`{"key": "value"}`）
2. Python dict 转换（单引号 → 双引号，`True/False/None` → `true/false/null`）
3. 宽松正则提取（处理字段值内含有嵌套单引号等不规范格式）
